# Bootcamp Excel com Inteligência Artifical - DIO
Repositório para arquivos de projetos do Bootcamp: Excel com Inteligência Artifical da DIO

## Projeto 1: Ferramenta para controle de investimentos
Planilha para auxiliar o usuário com investimentos de Fundos Imobiliários. A proposta é de uma ferramenta interativa onde o usuário possa ter previsibilidade de retorno e receber as melhores opções de divisão de investimento de acordo com seu perfil de investidor.
